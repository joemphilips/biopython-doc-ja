4.3.11 SWISS-PROT

SProt スキャナは以下のようなイベントを生成します: 
record
    identification
    accession
    date
    description
    gene_name
    organism_species
    organelle
    organism_classification
    reference_number
    reference_position
    reference_comment
    reference_cross_reference
    reference_author
    reference_title
    reference_location
    comment
    database_cross_reference
    keyword
    feature_table
    sequence_header
    sequence_data
    terminator

KeyWList スキャナは以下のようなイベントを生成します: 

header
keywords
    keyword
footer
    copyright
