4.4.3.3 使用例

データベース内の残基数が，以下のような空白区切りの形式でファイルに 入っているとします (この例では 3 つのアルファベットから成っています): 

A   35
B   65
C   100

データの読み込みは FreqTable.read_count(file_handle) で行います．

同じ内容の期待度数ファイルは以下のようになります: 

A  0.175
B  0.325
C  0.5

一方，残基の頻度や数は辞書でも渡せます． (3 文字アルファベットの場合の) 残基数辞書は以下のようになります: 

{'A': 35, 'B': 65, 'C': 100}

この残基数データから，'C' の頻度は 0.5 ，'B' は 0.325，'A' は 0.175 で，母数は A, B, C の合計値 200 になります．

同じデータを頻度辞書で表すと以下のようになります: 

{'A': 0.175, 'B': 0.325, 'C': 0.5}

合計は 1 になっていますね．

辞書型を引数に渡す場合，残基数の辞書なのか頻度の辞書なのかを 指定せねばなりません．そのため，FreqTable クラスの コンストラクタには二つの引数，辞書自体と，辞書の内容を表す シンボルを指定する必要があります．シンボルは FreqTable.COUNT またはFreqTable.FREQ で，それぞれ残基数と頻度を示します．

以下のどれを使っても，頻度テーブル (ftab) を作成できます: 

>>> from SubsMat import *
>>> ftab = FreqTable.FreqTable(my_frequency_dictionary,FreqTable.FREQ)
>>> ftab = FreqTable.FreqTable(my_count_dictionary,FreqTable.COUNT)
>>> ftab = FreqTable.read_count(open('myCountFile'))
>>> ftab = FreqTable.read_frequency(open('myFrequencyFile'))
