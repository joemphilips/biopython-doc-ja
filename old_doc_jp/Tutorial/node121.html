5.1.2 Windows 版

Windows 版には通常，よくできた GUI ベースのインストーラがついて来て， 必要なコンポーネントを正しい場所にインストールしてくれます． この手のインストーラは，Distutils を使ってかなり簡単に 作成できます．

まず，自分の Windows コンピュータで動き，C コンパイラを持っており， そのコンパイラで Biopython をコンパイルしてインストールできねば なりません．(やり方は Biopython インストールガイドを参照してください)．

C コンパイラを設定してあれば，インストーラの作成は以下のように するだけです: 

python setup.py bdist_wininst

これで Windows 版インストーラが手に入ります．おめでとう!
