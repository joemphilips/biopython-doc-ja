A.1 ハンドル (handle) とは何か

このドキュメントでは，全体を通してハンドルという言葉をよく使います． この言葉は (少なくとも私の考えでは) ややまぎらわしい言葉でも あります．基本的に，ハンドルはテキスト情報への ``ラッパ'' と 考えてください．

ハンドルを使うと，素のテキスト情報に比べて，(少なくとも) 以下のような 利点があります:

  1. ハンドルを使うと，様々な方法で保存されている情報を標準的な ひとつのやり方で扱えるようになります．テキスト情報には，ファイルに 入っているもの，メモリ上に記憶されているもの，どこか遠隔地のウェブ サイト上にあるもの，など様々ありますが，ハンドルを使えばこれら全ての 情報を共通のひとつのやり方で扱えます．

  2. ハンドルを使うと，テキスト形式の情報を一気に全て読むのではなく， 逐次的に読み出せます．これは，巨大なテキストファイルを扱っていて， 全てを読み込まなければならないのにメモリ上には乗り切らないといった 場合にとても重要な機能です．

ハンドルを使うと，テキスト情報の読み込み (ファイルからの読み込み)， および書き出し(ファイルへの書き込み) を扱えます． 読み込み用 (``read'') ハンドルの場合，よく使う関数は read です．この関数はハンドルから全てのテキスト情報を読み出します． また，readline を使うと，一度に一行づつ情報を読み出します． 書き出し (``write'') ハンドルでは，write をよく使います．

ハンドルのもっとも一般的な用途はファイルからの読み出しです． この場合，ハンドルを作成するには python の組み込み関数 open を使います: 

>>> handle = open("m_cold.fasta", "r")
>>> handle.readline()
">gi|8332116|gb|BE037100.1|BE037100 MP14H09 MP Mesembryanthemum crystallinum cDNA 5' similar to cold acclimation protein, mRNA sequence\n"

ハンドルは， Biopython で様々なパーザに情報を渡すときに使われます．
