A.1.1 文字列からハンドルを生成する

文字列中に入った情報からハンドルを生成しておくと便利な ことがあります．以下の例では，Python 標準モジュールのcStringIO を使って文字列からハンドルを生成する方法を示しています: 

>>> my_info = 'A string\n with multiple lines.'
>>> print my_info
A string
 with multiple lines.
>>> import cStringIO
>>> my_info_handle = cStringIO.StringIO(my_info)
>>> first_line = my_info_handle.readline()
>>> print first_line
A string

>>> second_line = my_info_handle.readline()
>>> print second_line
 with multiple lines.
