1.1 Biopython とは

Biopython プロジェクトは，無料で利用できる計算機分子生物学ソフトウェアを Python (http://www.python.org) で開発している人々の 国際的なつながりです．Biopython のウェブサイト， http://www.Biopython.org では，モジュールやスクリプトといった オンラインの情報， Python ベースの生命科学研究向けソフトウェアを開発 している人たちへのウェブリンクを提供しています．

Biopython にかかわる人たちは，基本的に Python でのプログラム作成が好きな 人々です．彼らは高品質で再利用できるモジュールやスクリプトを作成することで Python のバイオインフォマティクスへの応用を可能な限り簡単にしたいと 考えています．
