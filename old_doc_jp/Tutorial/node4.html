1.1.1 Biopython パッケージの内容

Biopython の主要なリリースには以下のような様々な機能があります:

  • バイオインフォマティクス関連のファイルを解析して， Python で利用 できるデータ構造に変換する機能．以下のフォーマットをサポートしています:
    • Blast 出力 - スタンドアロンプログラム，および WWW 上の Blast
    • Clustalw
    • FASTA
    • GenBank
    • PubMed および Medline
    • Expasy ファイル, Enzyme, Prodoc や Prosite
    • SCOP の 'dom' ファイルや 'lin' ファイル
    • Rebase
    • UniGene
    • SwissProt

  • サポートされているファイル形式では，レコード単位での反復処理を 行ったり，辞書インタフェースでアクセスしたりできます．

  • 広く利用されているバイオインフォマティクス関連のサービスを扱う ためのコード．例えば以下のサービスを扱えます:

    • NCBI - Blast, Entrez および PubMed サービス
    • Expasy - Prodoc および Prosite エントリ

  • 広く使われているバイオインフォマティクスプログラムへの インタフェース．以下のプログラムへのインタフェースがあります:

    • NCBI によるスタンドアロン Blast プログラム
    • Clustalw アラインメントプログラム

  • 配列自体，配列の id ，配列の機能 (feature) を扱える配列データクラス．

  • 塩基配列の転写や翻訳，質量計算といった配列に対してよく行われる 操作を実行するためのツール．

  • k-NN (k Nearest Neighbors) 法やNaive Bayes 法，SVM (Support Vector Machines) 法を使ってデータのクラスタリングを行うためのコード．

  • 代替行列 (substitution matrix) の作成や操作を含むアラインメント 処理を扱うためのコード．

  • 並列化可能なタスクを別々のプロセスに分割しやすくするためのコード．

  • 基本的な配列の操作，塩基配列の翻訳，BLAST処理などを行うための GUI ベースのプログラム．

  • モジュールの利用法に関するドキュメントやヘルプ，オンライン上の Wiki ドキュメントや Web サイト，メーリングリスト．

  • BioCorba インタフェース標準を利用した，Bioperl や Biojava プロジェクトといった他言語との協調 (Biopython-corba モジュールから 利用できます)．

ここまで読んだあなたは，きっと Biopython をダウンロードして 使い始めるだけの理由をいくつも見つけたでしょうね！
