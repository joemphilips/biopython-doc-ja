2. クイックスタート - Biopython でできること

この節の目的は，読者が手っ取り早く Biopython を使い始められるようにして， Biopython でできることとそのやり方についての概要を伝えることにあります． 本節の全ての例題は，読者が既に Python を使うときの一般的かつ実践的な知識を 多少持っていると想定して書かれています． Python についての知識をもう少し 磨いておくべきだと思うなら，Python の本家 Web サイト (http://www.python.org/doc/) に行けば，無料で読める 入門向けドキュメントが少しあります．

計算機上で扱える生物学関連の作業の多くには，インターネット上のデータベース に対する接続が含まれています．従って，例題のいくつかについては，正しく 動作させるためにインターネットへの接続が有効になっている必要があるでしょう．

さて，それはともかくとして，Biopython で何を実行できるかについて議論 しましょう．
