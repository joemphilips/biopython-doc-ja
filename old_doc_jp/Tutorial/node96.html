4.3.5 コンシューマ 

class Consumer:
    # イベントハンドラ

コンシューマには，イベントを処理するためのメソッドを入れておきます． メソッドの名前は，コンシューマが処理するイベントの名前になります． info イベントの場合には，イベントに関連した情報の入ったデータ行が 渡されます． section イベントの場合には何も渡されません．

自分のアプリケーションとは関係ないイベントは無視してかまいません． こうしたイベントのメソッドは実装しないでおきます．

コンシューマは，Consumer クラスから導出しなければなりません．

以下に例を示します: 

class FASTAConsumer(Consumer):
    def title(self, line):
        # タイトル行を処理する
    def sequence(self, line):
        # 配列情報の各行を処理する
    def begin_sequence(self):
        # 配列情報の開始部分
    def end_sequence(self):
        # 配列情報の終了部分
